# 🔢 Simple Calculator App

A basic calculator web app built using **HTML, CSS, and JavaScript** that performs standard arithmetic operations such as:

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division
